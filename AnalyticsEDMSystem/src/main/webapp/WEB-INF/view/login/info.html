EDM Analytic System

Info

  Приложение EDM Analytics System разработано для
ведения аналитики технологического процесса электроэрозионной
обработки.
  Цель разработки - повышение качества продукци, изготваливаемой
посредством электроэрозионной обработки.
  Приложение разработано на языке Java 9, с использованием фреймфорка
Spring.
Приложение имеет архитектуру MVC, разработано по методологии REST.

Стек технологий:

  • Java 9
  • Spring 5.2.1 RELEASE
  • MySQL
  • Hibernate ORM
  • Tomcat 9
  • HTML
  • CSS
